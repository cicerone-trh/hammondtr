First, I'd like to talk about this site a little bit. I decided to finally hack something together and decided it would be very simple. I wanted to make something without using any fancy front-end stuff; I understand that there are prettier, more impressive things, but I put this off long enough. I am currently working on a web app called "Cicerone" that is a sort of personal historian. I feel like posting GitHub repositories is one way of expressing your ability, but it heavily relies on the reader to have understand the individual's tacit understanding. Plus, I want to be able to record little things, like working through an essay or maybe a framework without actually creating anything substantial enough to create a repository. When I have hacked together a decent enough prototype, I'll include a link.
